# OpenCV-Demos

Examples to use OpenCV with khadas SBCs.

Compatible with VIM3/VIM3L/VIM4/Edge2

### Setup OpenCV

Install OpenCV Libraries

```bash
sudo apt update
sudo apt install libopencv-dev python3-opencv
```
